# Groq-Native Pipeline — Agents for Data Quality
## The "Deterministic-First" Architecture

This setup runs entirely on your local machine (or any environment with Python).
It uses Groq's free-tier API for LLM inference — no GPU required, no model download,
no session expiry. The LLM is invoked only where a deterministic algorithm cannot
make a decision: outlier classification and ambiguous remediation cases.

```

├── README.md
├── main.ipynb          ← THE single source of truth
```

## Phase 3 — Deterministic Tool Library

Pure Python functions that return structured dictionaries:
`{issues, severity, evidence, suggested_fix}`.
These are the agents' precise mathematical eyes — they provide hard evidence
so the LLM reasons on facts rather than raw CSV content.

**Design principle:** each tool is stateless, side-effect-free, and
receives only the DataFrame + dataset name. Tools that modify data
(remediation) receive and return the DataFrame explicitly.

## Phase 5 — Agent Definitions (LangGraph + Groq)

**Core design principle:** deterministic tools do the work.
The LLM is called only in two cases:
1. **AnomalyDetectionAgent** — to classify each IQR-flagged candidate as
   genuine error vs. legitimate extreme value, using row context.
2. **RemediationAgent** — for ambiguous corrections where a deterministic
   rule alone is insufficient (e.g. imputing a missing categorical field
   from row context). For all clear-cut cases, the fix is applied directly.

The LLM decides but never executes — a deterministic tool always performs
the actual DataFrame modification. Every LLM decision is recorded in the
correction log with its full reasoning.

```python
import operator
from typing import TypedDict, Optional, Annotated
from langchain_core.tools import tool
from langchain_core.messages import SystemMessage
from langgraph.prebuilt import create_react_agent
from langgraph.graph import StateGraph, END

# ── Shared state ──────────────────────────────────────────────────────────────

SHARED_DF: pd.DataFrame = None   # lives outside LangGraph state to avoid
                                  # repeated serialization on every tool call

class AgentState(TypedDict):
    config:               dict
    schema_report:        Optional[dict]
    completeness_report:  Optional[dict]
    consistency_report:   Optional[dict]
    anomaly_report:       Optional[dict]
    fixed_df_json:        Optional[str]
    correction_log:       Optional[list]
    reliability_score:    Optional[float]
    score_breakdown:      Optional[dict]
    iterations:           int
    current_agent:        str
    errors:               Annotated[list, operator.add]

# ── LangGraph tool wrappers (read-only, return JSON strings) ──────────────────

@tool
def get_schema_report(dummy: str) -> str:
    """Runs schema validation on the current dataset. Returns JSON report."""

@tool
def get_completeness_report(dummy: str) -> str:
    """Runs completeness analysis. Returns JSON with null counts and sparse columns."""

@tool
def get_consistency_report(dummy: str) -> str:
    """Runs format and cross-column consistency checks. Returns JSON report."""

@tool
def get_outlier_candidates(dummy: str) -> str:
    """Returns IQR-flagged outlier candidates with row context for LLM classification."""

@tool
def get_categorical_frequencies(column: str) -> str:
    """Returns value frequency distribution for a categorical column."""

# ── Remediation tools (modify SHARED_DF in place) ────────────────────────────

@tool
def impute_numeric_median(column: str) -> str:
    """Imputes null numeric values with column median. Returns correction log entries."""

@tool
def impute_categorical_mode(column: str) -> str:
    """Imputes null categorical values with column mode. Returns correction log entries."""

@tool
def impute_categorical_llm(params_json: str) -> str:
    """LLM-assisted imputation for ambiguous categorical nulls.
    Input: JSON {column, context_columns}.
    LLM infers value from row context; validates against known domain.
    Falls back to mode if inferred value is outside valid domain.
    Returns correction log entries with action 'impute_llm' or 'impute_mode_fallback'."""

@tool
def apply_llm_outlier_decision(params_json: str) -> str:
    """Applies a remediation action previously decided by the LLM for an outlier.
    Input: JSON {row_index, column, action, reason}.
    Action is one of: 'clip', 'keep', 'null'.
    LLM decides, this tool executes. Returns correction log entry."""

@tool
def remove_duplicates(key_columns_json: str) -> str:
    """Removes duplicate rows keeping first occurrence. Returns correction log entries."""

@tool
def normalize_date_format(column: str) -> str:
    """Normalizes all dates in column to most frequent valid format. Returns correction log."""

@tool
def clip_outliers_iqr(column: str) -> str:
    """Clips values outside IQR boundaries. No row deletion. Returns correction log."""

@tool
def get_current_df_json(dummy: str) -> str:
    """Returns current SHARED_DF as JSON for post-fix verification."""

# ── Agent nodes ───────────────────────────────────────────────────────────────

def schema_validation_agent(state: AgentState) -> dict:
    """Runs deterministic schema tools. No LLM call — violations are binary."""

def completeness_agent(state: AgentState) -> dict:
    """Runs deterministic null and sparse-column tools. No LLM call."""

def consistency_agent(state: AgentState) -> dict:
    """Runs deterministic format and cross-column tools. No LLM call."""

def anomaly_detection_agent(state: AgentState) -> dict:
    """Runs IQR tool to get outlier candidates, then calls LLM to classify
    each candidate as genuine error vs. legitimate extreme value."""

def remediation_agent(state: AgentState) -> dict:
    """Receives all four reports. Applies fixes deterministically for clear cases.
    Calls LLM only for ambiguous decisions (imputation from row context,
    outlier disposition). LLM decides; tools execute."""
```

## Phase 6 — LangGraph Feedback Loop

Wire the nodes into a graph with a supervisor check after remediation.
The supervisor reruns the full pipeline if critical issues remain
or the post-remediation score is below threshold, capping at 3 iterations.

```python
SCORE_THRESHOLD = 80.0   # minimum acceptable reliability score (0–100)

def supervisor_check(state: AgentState) -> str:
    """Routing function — decides whether to end or rerun the pipeline."""
    if state["iterations"] >= 3:
        return "end"

    score = state.get("reliability_score") or 0.0
    reports = [state.get(f"{k}_report") or {} for k in
               ["schema", "completeness", "consistency", "anomaly"]]
    critical_remaining = sum(
        1 for r in reports
        for issue in r.get("issues", [])
        if issue.get("severity") == "critical"
    )

    if score < SCORE_THRESHOLD or critical_remaining > 0:
        return "schema"   # rerun from first agent
    return "end"

def build_pipeline():
    graph = StateGraph(AgentState)

    graph.add_node("schema",       schema_validation_agent)
    graph.add_node("completeness", completeness_agent)
    graph.add_node("consistency",  consistency_agent)
    graph.add_node("anomaly",      anomaly_detection_agent)
    graph.add_node("remediation",  remediation_agent)
    graph.add_node("report",       report_generator)

    graph.set_entry_point("schema")
    graph.add_edge("schema",       "completeness")
    graph.add_edge("completeness", "consistency")
    graph.add_edge("consistency",  "anomaly")
    graph.add_edge("anomaly",      "remediation")
    graph.add_edge("remediation",  "report")

    graph.add_conditional_edges(
        "report",
        supervisor_check,
        {"schema": "schema", "end": END}
    )

    return graph.compile()

pipeline = build_pipeline()
print("✅ Pipeline compiled — feedback loop active (max 3 iterations)")
```
